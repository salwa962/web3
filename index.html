Web Applications

What is a Web Application

A web application is software that runs in a web browser. Businesses use web applications to exchange information and deliver services remotely. These applications connect with customers conveniently and securely through features such as shopping carts, product searches, instant messaging, and social media newsfeeds. Web applications enable access to complex functionalities without requiring the installation or configuration of additional software.

How do Web Applications interact with Back-End Databases

  1. Client Request:
    The user performs an action (e.g., filling out a form or clicking a button) that generates a request sent to the server.
  2. Server Processing:
    The server receives and processes the client request. This may involve formulating a database query to either retrieve or modify data.
  3. Database Query:
    The server-side script constructs a query to the database. This can be a read operation (SELECT) to fetch data or a write operation (INSERT, UPDATE, DELETE) to change data.
  4. Database Response:
    The database executes the query and sends the results back to the server. For read operations, it returns the requested data. For write operations, it confirms that the data manipulation was successful.
  5. Server Response:
    The server processes the database's response and generates a response to be sent back to the client. This might include the requested data or a confirmation message.
  6. Client Update:
    The client-side script receives the server's response and updates the user interface accordingly, displaying new information or confirming the success of an action. This interaction ensures that the web application remains dynamic and responsive, providing users with up-to-date data and interactive features.

Examples of Intranet and Internet Use Cases in Businesses

An intranet content management system (CMS) supports document management and data storage, information publishing, customizable sitemap and navigation, integration with third-party tools, a collaborative workspace for employees, and a virtual meeting space. Internet use cases include e-commerce platforms for online shopping and payments, customer relationship management (CRM) systems for managing customer data and sales, marketing and advertising tools for social media management and SEO, and online collaboration tools for project management and video conferencing.
